Project Page

The project is developed using Godot Engine 4, using C# and GDScript as the main code. The project is described to be a turn-based Roguelite game, with the visual graphics of a 2D game. The game uses ATB turn system with a traditional "take your move" turn process, with the niche of the game to have the resource be centralized to everything.

In development, OOP programming is practiced with the implementation of composition and inheritance. A clear distinction of uses cases are identified and then adapted accordingly in which a component can be an abstract class of an base object to be processed in the general handler, but have the possibility to extend functionality through composition.

The code also involves JSON data file processing, which the game instance reads the file and converts it into an object instance. This enables the game to possess a database system despite not having one handled by a server software like MySQL.
